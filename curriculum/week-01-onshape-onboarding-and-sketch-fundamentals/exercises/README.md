# Week 1 — Exercises

Short, focused sketching drills you perform **inside OnShape**. There is no code — each exercise is a precise, numbered modeling walkthrough. Open OnShape in one tab, this file in another, and *do every step at the keyboard*. Sketching is a motor skill; reading about it is not enough.

Do them in order. Later ones assume the muscle memory the earlier ones build.

## Index

1. **[Exercise 1 — First Document and sketch](exercise-01-first-document-and-sketch.md)** — create a Document, set units, navigate, then draw and **fully define** a simple plate with two holes. Watch geometry update parametrically. (~35 min)
2. **[Exercise 2 — Constraint drills](exercise-02-constraint-drills.md)** — six numbered drills, each isolating one or two constraints (horizontal/vertical, parallel, perpendicular, equal, concentric, symmetric). Each ends fully defined with the **minimum** dimensions. (~45 min)
3. **[Exercise 3 — Arc and slot lever profile](exercise-03-arc-and-slot-profile.md)** — tangent arcs, the slot tool, concentric circles, and a construction centerline combined into one fully-defined lever. (~45 min)

## How to work the exercises

- Read the step. Skim, don't memorize — you'll do it, not recite it.
- **Drive the geometry yourself.** Do not copy a finished Document; build each sketch by hand. The point is the reflex.
- After each drill, run the **drag test**: try to drag a line or point. If anything moves, you're not done — find the free degree of freedom.
- **Watch the color.** Every exercise must end with **all geometry black** and the OnShape status bar reading **"Fully defined."** A blue line is a bug this week.
- Use the **right tool** so constraints come for free (rectangle tool, tangent-arc tool, slot tool). Read the inference glyphs as you draw.
- Turn on **Show constraints** before you finish, and *look* at what's holding the geometry together.

## Done-with-the-week checklist

Tick these off as you complete the exercises:

- [ ] I can create a Document, name it, and set its workspace units.
- [ ] I can rotate, pan, zoom, and "view normal to" a plane fluidly with a mouse.
- [ ] I anchored a first sketch to the **origin** and understand why.
- [ ] I fully defined a sketch and confirmed **all geometry is black**.
- [ ] I changed a driving dimension and watched the geometry rebuild parametrically.
- [ ] I applied each of: coincident, horizontal/vertical, parallel, perpendicular, tangent, equal, concentric, symmetric, midpoint.
- [ ] I built a profile using the **minimum** number of dimensions, leaning on constraints.
- [ ] I used a **construction** centerline and a **symmetric** constraint.
- [ ] I used the tangent-arc and slot tools.
- [ ] I created a named **Version** of my Document.
