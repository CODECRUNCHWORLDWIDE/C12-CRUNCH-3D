# Week 1 — Challenges

The exercises drill basics. **Challenges stretch you.** This week's challenge takes 60–90 minutes and produces a sketch you can keep in your portfolio Document — a clean reproduction of a real dimensioned engineering drawing.

## Index

1. **[Challenge 1 — Gasket profile from a dimensioned drawing](challenge-01-gasket-profile.md)** — read a dimensioned 2D engineering drawing of a flat gasket and reproduce it as a **single fully-defined sketch** using the **minimum** number of dimensions, leaning on constraints instead of over-dimensioning. (~90 min)

## How it's assessed

Challenges are optional — you can pass the week without them — but this one is the truest test of the week's skill, because it forces the discipline the exercises only suggest. It's assessed on four things:

1. **Correctness** — your sketch matches the target drawing's geometry and dimensions.
2. **Fully defined** — all geometry black, status bar "Fully defined," no blue, no conflicts.
3. **Minimum dimensions** — you used constraints (`equal`, `concentric`, `symmetric`, `tangent`, `parallel`, `perpendicular`) wherever a relationship could replace a number. We literally count your dimensions against a target.
4. **Robustness** — changing one key driving dimension rebuilds the whole profile cleanly, proving the constraint scheme is sound.

A profile that *looks* right but is held together by 18 dimensions and zero relationships **fails** this challenge even if every number is correct. The whole point is that constraints encode intent and dimensions specify only what genuinely varies. If you do this challenge well, the mini-project master sketch will feel easy — and so will all five weeks after it.
